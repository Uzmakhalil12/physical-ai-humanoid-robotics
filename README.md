# Physical AI and Humanoid Robotics

This repository contains the documentation for the Physical AI and Humanoid Robotics Project, built with Docusaurus.

## Features

- Comprehensive documentation for the Physical AI and Humanoid Robotics Project
- Physical AI & Humanoid Robotics textbook content
- Interactive learning tools
- Multilingual support (English and Urdu)
- Responsive design for all devices

## Installation

```bash
# Navigate to the docusaurus directory
cd docusaurus

# Install dependencies
npm install

# Start the development server
npm start
```

## Build for Production

```bash
# Build the static website
npm run build
```

## Deployment

This site can be deployed to:
- GitHub Pages
- Vercel
- Any static hosting service

For Vercel deployment, the configuration is already provided in `vercel.json`.
